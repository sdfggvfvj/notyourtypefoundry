# notyourtypefoundry

notyourtypefoundry is an independent experimental type foundry based in Rotterdam, NL, founded by Xiaoyuan Gao in 2021.

This repository contains the **Free To Use BUT Be Nice License** — the license that applies to free typefaces released by notyourtypefoundry.

---

## Free To Use BUT Be Nice License

Some of our typefaces are free to use under specific conditions. This license is designed to keep type accessible to individuals, students, small businesses, and cultural institutions — while ensuring that large commercial entities obtain a proper license.

The full license text is available here: [LICENSE.md](./LICENSE.md)
